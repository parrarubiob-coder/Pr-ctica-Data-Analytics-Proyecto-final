# 📊 Olist E-Commerce Dashboard

An end-to-end Data Analytics project using the **Brazilian Olist E-Commerce Dataset**. The project combines **Python** for data cleaning and preparation with **Power BI** for interactive dashboard development, providing business insights into sales, logistics, customers, sellers, and product performance.

---

## 🚀 Project Highlights

- ✅ Cleaned and integrated **9 relational datasets** into a single analytical model.
- ✅ Built a **4-page interactive Power BI dashboard** with more than **25 KPIs and visualizations**.
- ✅ Performed data cleaning, feature engineering, and data modeling using **Python (Pandas)**.
- ✅ Created an analytical dataset optimized for Business Intelligence.
- ✅ Identified key business insights about sales performance, logistics, customer behavior, and product categories.

---

# 🎯 Objectives

The main objective of this project is to transform raw e-commerce data into meaningful business insights by:

- Cleaning and transforming the original datasets.
- Building a reliable analytical data model.
- Designing an interactive Power BI dashboard.
- Supporting business decision-making through KPIs and visualizations.

# 🐍 Python Data Preparation

The Python notebook performs the complete **ETL (Extract, Transform, Load)** process.

### Data preparation includes:

- Importing all Olist datasets.
- Data exploration and quality assessment.
- Checking and correcting data types.
- Removing duplicates.
- Handling missing values.
- Standardizing column names.
- Translating product categories into English.
- Cleaning the geolocation dataset.
- Merging all datasets into a single analytical table.
- Exporting cleaned datasets to the **processed** folder.

The final dataset was optimized for Power BI analysis.

# 📈 Power BI Dashboard

The dashboard consists of **four interactive pages**.

---

## 📊 1. Sales Overview

Provides an executive overview of business performance.

### KPIs

- Total Revenue
- Total Orders
- Average Order Value
- Average Shipping Cost
- Unique Customers

### Visualizations

- Revenue by State
- Monthly Revenue & Orders
- Top Product Categories
- Payment Methods

---

## 🚚 2. Logistics Performance

# 📊 Key Insights

Some of the insights obtained from the analysis include:

- São Paulo dominates the Brazilian e-commerce market, concentrating the highest number of customers and sellers.
- Credit cards represent the most frequently used payment method.
- More than 97% of orders are successfully delivered.
- **Health & Beauty** is the highest revenue product category.
- Customer demand and seller distribution vary significantly across Brazilian states.
- Some states show a higher customer-to-seller ratio, suggesting potential opportunities for expanding the seller network.
